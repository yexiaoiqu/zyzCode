## 运行时报错汇总及原因：
### 1. SyntaxError: invalid syntax → 第158行
原因 ：我之前修改换行时，括号拆分错误，导致左括号没有正确闭合，Python解析失败。
 修复 ：补全缺失的右括号。

### 2. NameError: name 'W' is not defined → 第251行
原因 ：代码直接使用 W ，但实际上 W 是 self.p.W_swing ，缺少 self.p. 前缀，作用域不对找不到变量。
 修复 ：改为 self.p.W_swing 。

### 3. ValueError: all the input array dimensions except for the concatenation axis must match exactly → np.block
原因 ：矩阵分块维度计算错误。状态是10维，原来的分块方式导致维度不匹配无法拼接。
 修复 ：正确分块 → 3维位置 + 3维速度 + 4维摆动 = 10维，分别分块拼接。

### 4. ValueError: x and y must have same first dimension → matplotlib绘图
原因 ： kappa_e_history 历史数据累积，多次仿真后长度越来越长（第一次400 → 第二次800 → 第三次1200...），但 time 长度始终是 400，导致长度不匹配。
 修复 ：

- 在每次仿真开始前 强制清空 自适应权重历史
- 绘图时增加保护 n_points = min(len(time), len(kappa_e)) ， 强制截断 保证长度一致
### 5. FileNotFoundError: No such file or directory → 保存图片
原因 ：原来用的是 Linux 路径 /mnt/user-data/outputs/ ，Windows 系统上这个路径不存在。
 修复 ：改为相对路径 outputs/ ，并添加 os.makedirs('outputs', exist_ok=True) 确保输出目录存在。

## 现在所有运行时错误都已修复，可以正常运行了。
